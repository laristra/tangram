For a given cell, computes per-material volume of symmetric difference between reference polyhedra with that material and reconstructed MatPoly's from the CellMatPoly object.

Parameters
[in]reference_mat_polysfor every cell material, the collection of reference single-material polyhedra containing that material
[in]ref_mat_idsIDs of cell materials
[in]ref_mat_volreference volumes of cell materials
[in]result_ptrpointer to CellMatPoly object containing the results of interface reconstruction for this cell
[out]mat_sym_diff_volper-material volumes of symmetric difference
[in]convex_matpolysIf material polyhedra in CellMatPoly are non-convex, this flag should be set to true in order to decompose them into tetrahedrons
