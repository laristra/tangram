Detailed Description

template<template< class, int, class, class > class CellInterfaceReconstructor, int Dim, class Mesh_Wrapper, class MatPoly_Splitter = void, class MatPoly_Clipper = void>
class Tangram::Driver< CellInterfaceReconstructor, Dim, Mesh_Wrapper, MatPoly_Splitter, MatPoly_Clipper >

Driver provides the API for reconstructing interfaces in multimaterial cells and answering queries about said interfaces.

Template Parameters
CellInterfaceReconstructorA class that can reconstruct the pure material polygons in a cell given the volume fractions in the cell (and if needed its neighbors)
Mesh_WrapperA lightweight wrapper to a specific input mesh implementation that provides certain functionality.
